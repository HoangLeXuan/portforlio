Gelp 2.0 – AI-Powered Food Recommendation Chatbot

A conversational AI chatbot that delivers personalized food recommendations based on user preferences and demographics. Built using natural language processing and designed with user interaction in mind, Gelp 2.0 demonstrates practical application of machine learning in a real-world, user-friendly format.

What is it?

Gelp 2.0 is an AI-powered chatbot that provides personalized food recommendations based on user preferences and demographics. Built using natural language processing techniques and designed for conversational interaction, it simulates a dialogue with users to understand their tastes and dietary needs before generating intelligent, user-centric suggestions.

Objective

Creating the first chat bot, and learning developing an AI chatbot capable of understanding user input and offering tailored food recommendations.

Value Proposition

This project is a fun user-product development, particularly in real-time applications. It demonstrates proficiency in modern JavaScript frameworks, WebSocket technology, and user experience design.
